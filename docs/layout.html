布局

{% include anchor.html name="Container" %}

Baobo.css 需要为页面内容和阵列系统包裹一个 .container 容器。每一个容器的宽度都基于一个固定的公式：$device - (2 * $gap)。$gap 的默认值为 32px，当然你可以修改它。

This is how the container will behave:

  • $desktop 最宽为 960px
  • $widescreen 最宽为 1152px
  • $fullhd 最宽为 1344px
{% highlight html %}{{container_example}}{% endhighlight %} {% include anchor.html name="Level" %}

Baobo.css 同样提供了一个水平方向的布局容器，几乎可以放入任何组件。

  • level
    • level-left
    • level-right
      • level-item

在水平布局系统中，你可以插入几乎任何想要的内容：标题、按钮、文本输入框等。无论你在其中放置什么组件，它们都将始终处于垂直居中。

默认情况下，由于空间的限制，水平布局在移动设备上是垂直的。如果你希望水平布局在移动设备也上保持水平，请在 .level 的容器上添加 is-mobile。
