阵列系统

Baobo.css 提供了一套响应式、移动设备优先的流式阵列布局系统。

{% include anchor.html name="列布局" %}

用 Baobo.css 创建一个基于列的布局非常简单：

  1. 添加一个 columns 作为容器
  2. 根据需要添加 column

值得注意的是，在默认状态下无论有多少个 column，它们的宽度都是一致的。

{% include anchor.html name="列偏移" %}

使用 .is-offset-x 类可以将列向右侧偏移。这些类实际是通过使用 * 选择器为当前元素增加了左侧的边距（margin）。例如，.is-offset-4 类将 .is-8 元素向右侧偏移了4个列（column）的宽度。

{% include anchor.html name="窄列" %}

如果您想要一列仅占用它所需的空间，而另一列将填满剩余的空间的话，请使用 is-narrow 修饰符。当然针对不同设备上的特殊情况，你也可以是单独设定它们：

  • is-narrow-mobile
  • is-narrow-tablet
  • is-narrow-desktop
